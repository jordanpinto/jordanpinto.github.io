How I started programming

Posted on February 2, 2018

Hi World,
I'm so excited to start this blog to let everyone know about my journey and how it started. Growing up, I didn't really care about how computers worked or the "code behind the curtain". In fact, I didn't even think about learning to code until I was about 28. I had an epiphany one day that learning to code was going to be a very important skillset - especially when I searched for jobs and read about the increasely significant call for automation.

So I signed up at FreeCodeCamo and started to learn HTML, CSS, and Javascript because I really wanted to learn front-end web development and possibly do freelance web design!

Thanks for reading - see you next time!
